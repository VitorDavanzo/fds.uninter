# Landing Page Curitiba - Estudo de Front-end
Este projeto é um marco inicial nos meus estudos de **Análise e Desenvolvimento de Sistemas.** Trata-se de uma landing page informativa sobre a cidade de Curitiba, desenvolvida para exercitar a estrutura de documentos HTML, estilização e o uso de frameworks de mercado.

## Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.

* **CSS3:** Estilização personalizada através do arquivo estilo.css.

* **Bootstrap 3:** Framework utilizado para agilizar o layout responsivo, componentes de navegação (navbar) e painéis colapsáveis.

* **JavaScript & jQuery:** Utilizados para o funcionamento do Carrossel de imagens e interações de colapso do Bootstrap.

## Conceitos Praticados
* **Responsividade:** Uso de classes do Bootstrap como container, img-responsive e o sistema de grid para garantir que o site funcione em diferentes tamanhos de tela.

* **Componentes de UI:** Implementação de um Carrossel dinâmico para galeria de fotos e o componente Accordion (painéis que abrem e fecham) para a seção "Sobre nós".

* **Gerenciamento de Ativos:** Organização de diretórios para imagens, fontes e bibliotecas externas, garantindo que os caminhos (./img/, ./bootstrap/) estivessem corretos.
